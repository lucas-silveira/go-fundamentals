# Comandos básicos

Acessar as variáveis de ambiente:
```
go env
```

Compilar e executar um arquivo:
```
go run arquivo.go
```

Gerar um arquivo compilado:
```
go build arquivo.go
```

Baixar e instalar um pacote do Github:
```
go get -u github.com/caminho-do-pacote
```

Validar arquivo e expor erros:
```
go vet arquivo.go
```
